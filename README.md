# 📊 Madhav E-commerce Sales Dashboard | Power BI

## 📌 Project Title
**Madhav E-commerce Sales Dashboard | Power BI**

---

## 📝 Short Description
Developed an interactive Power BI dashboard to analyze Madhav E-commerce sales data. The dashboard provides valuable insights into sales performance, profit trends, customer behavior, product categories, and payment methods using dynamic filters and interactive visualizations, enabling data-driven business decisions.

---

## 🛠️ Tech Stack
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel
- Data Modeling
- Data Visualization

---

## 📂 Data Source
**Dataset:** Madhav E-commerce Sales Dataset (Excel)

The dataset includes:
- Orders
- Customers
- Products
- Categories & Sub-Categories
- States
- Payment Modes
- Sales Amount
- Quantity
- Profit
- Order Date

---

## ✨ Features
- 📊 Interactive dashboard with Quarter and State slicers
- 📈 Monthly Profit Analysis to identify trends and seasonal performance
- 🌍 State-wise Sales Analysis for regional performance comparison
- 🛍️ Category and Sub-Category Analysis based on sales quantity and profit
- 💳 Payment Mode Distribution and Customer-wise Sales Insights
